# 왜 다른 이메일 주소는 자동 승인을 해주지 않나요?
서울, 고려, 연세, 한양대학교 등에도 해외유학 지원 프로그램이 있는 것으로 알고 있습니다.
다만 종합대 특성상 인원에 비해 실 유학자 비율이 낮은 관계로, 가입 자동 승인은 하지 못하고 있습니다.
반달리즘이 일어날 경우, 관리자가 이를 확인하고 복구까지 얼마나 걸릴지 모르기 때문입니다.

자비로 서머세션을 다녀오신 경우에도 이에 대한 인증이 필요하기 때문에 수동으로 처리해드리고 있습니다.

# 수동 회원가입 방법
하기의 방법 중 편하신 방법으로 [enc2586@gmail.com](mailto:enc2586.com)에 메일 보내주시면 3일 내로 처리해드리겠습니다.

## A. berkeley.edu 메일을 통한 인증
1. 메일로 본인의 **a)개인 이메일 주소**, **b)berkeley 이메일 주소**를 보내주세요. (제목: `UCB위키 가입 신청 - type A`)
2. 관리자가 해당 **b)berkeley 이메일 주소**로 인증번호를 보내드립니다.
3. 해당 인증번호를 **a)개인 이메일 주소**에서 관리자에게 보내주세요. (제목: `UCB위키 가입 인증코드`)
4. 관리자가 확인 후 계정을 생성해드립니다.
> 보통 유학 종료 후 약 1년간, `@berkeley.edu` 계정은 유지됩니다.
개인정보 노출이 거의 없기에 이 방법을 추천합니다.

## B. 유학 서류를 통한 인증
1. 메일로 **I-20(개인정보 마스킹 필수)** 를 보내주세요. (제목: `UCB위키 가입 신청 - type B`)
	- PROGRAM START/END DATE, SCHOOL NAME은 보여야 합니다.
2. 관리자가 확인 후 해당 이메일로 계정을 생성해드립니다.
> I-20 위조는 사문서위조에 해당합니다! 설마... 하진 않으시겠죠?!
